使用说明

  • 代码高亮：编辑器会自动对 JavaScript 语法进行高亮显示
  • 智能提示：输入代码时会自动显示相关建议（如变量、函数、关键字等）
  • 自动补全：输入括号、引号等符号时会自动补全
  • 代码格式化：点击"格式化代码"按钮可自动美化代码格式
  • 本地存储：点击"保存到缓存"可将代码保存到浏览器本地存储，刷新页面后仍可恢复
  • 快捷键：支持常用快捷键（如 Ctrl+Space 触发提示，Ctrl+/ 注释代码等）
