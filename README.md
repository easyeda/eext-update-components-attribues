## 安全与回滚

- 强烈建议在首次运行时备份数据，确认变更后的内容符合预期后再保存，如果不符合预期，可以直接点击右上角关闭文档并选择“不保存”

## 故障排查

- 如果未匹配到任何元器件：
    - 检查器件属性是否正确设置
    - 检查目标库是否正确
    - 检查文档是否只读

本插件主要功能可分为两类：

1. 属性匹配与替换

- 根据制造商编号、料号、供应商编号或原理图元件的自定义属性值，在指定库中查找对应器件的属性值。
- 将匹配到的库中指定属性批量替换为用户指定的属性或属性值）。

2. 完整器件替换

- 通过制造商编号、料号、供应商编号或原理图自定义属性去匹配库中的对应器件。
- 找到匹配器件后可将原理图中的元件整体替换为库中对应的器件（包括属性、封装、引脚映射等）
